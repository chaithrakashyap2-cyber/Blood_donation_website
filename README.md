# Blood Donation Registration System

## Overview
This is a **web-based Blood Donation Registration system** designed to manage donor information and facilitate blood donation drives. Users can register themselves as donors, providing essential details like name, age, blood group, contact information, and preferred donation date. The system also includes forms for urgent blood requirements and searching available blood stocks.  

---

## Features

- **Donor Registration:** Collects donor details including name, age, blood group, contact info, and donation date.
- **Urgent Blood Requests:** Users can submit urgent blood requirements which are prioritized.
- **Search Blood Stock & Donors:** Quickly find available blood stocks and registered donors in a specific location.
- **Responsive Design:** Clean, user-friendly interface that works across devices.
- **Form Validation:** Ensures valid input for critical fields such as age, contact number, email, and date.
